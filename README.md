# 📱 TG WebApp Dashboard

Цей репозиторій містить фронтенд-частину веб-додатка для **Telegram WebApp**. Проєкт зосереджений на створенні сучасного інтерфейсу, дизайні кнопок та базових віджетів.

> **Основна мета:** Навчитися робити якісний веб-інтерфейс, який згодом буде підключений до Python-бота.

---

## 🛠 Структура проєкту

* 📂 `index.html` — Головна сторінка та каркас додатку.
* 🎨 `style.css` — Стилі сторінки та адаптивна верстка.
* ⚡ `script.js` — JavaScript для логіки кнопок та роботи віджетів.

---

## 🚀 Початок роботи

1.  **Клонувати репозиторій:**
    ```bash
    git clone [https://github.com/navigator2601/tg-webapp-dashboard.git](https://github.com/navigator2601/tg-webapp-dashboard.git)
    ```
2.  **Відкрити проект:** Скористайтеся VS Code або будь-яким зручним текстовим редактором.
3.  **Перегляд:** Відкрийте файл `index.html` у браузері для перевірки змін.

---

## 📋 План розробки (Tasks)

На даному етапі ми працюємо над:
- [ ] **Кнопки меню:** Створення зручної навігації.
- [ ] **Віджети:** Реалізація годинника та блоку погоди.
- [ ] **Адативність:** Коректне відображення на різних смартфонах.
- [ ] **Заглушки:** Тимчасові повідомлення для неактивних функцій.

### Схема функціоналу та заглушок

| Кнопка | Призначення | Очікувана поведінка |
| :--- | :--- | :--- |
| **Головна** | Основна сторінка | Виводить: "Ця кнопка веде на головну" |
| **Годинник** | Віджет часу | **Відображає поточний реальний час** |
| **Погода** | Віджет погоди | Виводить: "Тут буде погода, натисніть щоб оновити" |
| **Налаштування**| Налаштування | Виводить: "Налаштування будуть тут" |
| **Про додаток** | Інформація | Виводить: "Інформація про додаток" |

> **Приклад реалізації заглушки в `script.js`:**
> ```javascript
> document.getElementById('btn-weather').addEventListener('click', () => {
>     alert('Тут буде погода, натисніть щоб оновити');
> });
> ```
