# TG WebApp Dashboard

Простий веб-додаток для Telegram WebApp.

## 🎯 Мета

Навчитися створювати інтерфейс (UI) для Telegram WebApp:
- кнопки меню
- віджети
- базова логіка на JavaScript

## 📁 Структура проекту

- `index.html` – структура сторінки  
- `style.css` – стилі  
- `script.js` – логіка (годинник, кнопки)

## 🚀 Запуск

1. Відкрити проект у VSCode  
2. Відкрити файл `index.html` у браузері  
3. Вносити зміни та одразу перевіряти результат  

## 🧪 Функціонал

- ⏰ Годинник (працює)
- 📋 Меню (кнопки з обробкою кліків)
- 🔔 Повідомлення про натискання кнопок

## 📌 Завдання

Потрібно покращити:
- дизайн сторінки
- вигляд кнопок
- відображення інформації

Детальні задачі дивись нижче 👇

Завдання:

🔹 Рівень 1 (обов’язково)
1. Покращити дизайн
- зробити фон більш сучасним (градієнт)
- додати відступи між блоками
- вирівняти елементи

2. Покращити кнопки
- зробити hover ефект
- зробити плавну анімацію натискання
- додати іконки або стиль

👉 приклад:
.menu-btn:hover {
    background-color: #1f7ab8;
}

3. Статус повідомлення
Зараз це просто текст. Потрібно:
- зробити як окремий блок
- додати фон
- зробити “як повідомлення”

🔹 Рівень 2 (середній)

4. Замість alert зробити кастомне повідомлення
👉 щоб не було стандартного alert
Ідея:
- маленьке вспливаюче повідомлення (toast)

5. Анімація появи
- плавне завантаження сторінки
- fade-in для блоків

6. Адаптивність
- щоб нормально виглядало на телефоні
- перевірити різні розміри екрану

🔹 Рівень 3 (просунутий)
7. Розділення на “екрани”
Наприклад:
Головна
Погода
Профіль
(без перезавантаження сторінки)

8. Додати простий віджет погоди (заглушка)
Замість alert:
Погода: +18°C, ясно ☀️
