⚡ WebApp + Telegram Bot (без сервера)

Це інструкція як запустити WebApp через GitHub Pages і підключити його до Telegram.

🔧 КРОК 1: GitHub Pages

1. Відкрити репозиторій
2. Перейти: Settings → Pages
3. Обрати:
  • Branch: main
  • Folder: /root
4. Натиснути Save

Після цього з’явиться посилання:

https://username.github.io/repository-name/

✔ Це і є твій WebApp

🤖 КРОК 2: Telegram Bot

1. В Telegram знайти @BotFather
2. Ввести: /newbot
3. Придумати ім’я і username

✔ Бот створений

🔘 КРОК 3: Кнопка WebApp (БЕЗ КОДУ)

1. В BotFather:
/setmenubutton
2. Обрати бота
3. Текст кнопки:
Відкрити WebApp
4. Вставити URL:
https://username.github.io/repository-name/

✔ Тепер кнопка відкриває сайт

⚠️ ВАЖЛИВО

❌ НЕ ПРАЦЮЄ:

github.com/.../blob/...
raw.githubusercontent.com/...

✔ ПРАЦЮЄ:

github.io

🚀 РЕЗУЛЬТАТ

  • Відкриваєш бота
  • Натискаєш кнопку
  • Відкривається WebApp
